

# Переделка шапки сайдбара с логотипом

## Текущие проблемы
- Белая подложка вокруг маленького логотипа выглядит чужеродно на тёмном фоне
- Логотип слишком маленький (32x32px), теряется
- Компоновка «иконка + текст» не выразительна

## Новый вариант

Логотип занимает всю ширину сайдбара, без белой подложки. Компоновка — по центру, с отступами сверху/снизу и разделителем внизу.

```text
┌──────────────────────┐
│                      │
│   [логотип ~80%w]    │
│   Партнёрства        │
│──────────────────────│
│ УПРАВЛЕНИЕ ...       │
```

При collapsed сайдбаре — только логотип (уменьшенный до ~28px), без текста.

## Изменения

### `src/components/AppSidebar.tsx`
- Убрать белый квадрат (`bg-white rounded-md shadow-sm p-0.5`)
- Логотип: `w-full max-w-[140px] mx-auto` (expanded) / `h-7 w-7` (collapsed)
- Подпись «Партнёрства» — под логотипом, по центру, мелким шрифтом
- Добавить `border-b border-sidebar-border` под блоком для визуального отделения от меню
- Весь блок: `flex flex-col items-center px-4 py-4`
