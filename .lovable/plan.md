

# Убрать кнопку «Удалить» с видного места

## Что сейчас

Кнопка «Удалить» (красная, `variant="destructive"`) стоит прямо в заголовке рядом с названием партнёра/коллектива. Видна только админу (`isAdmin`), но бросается в глаза.

Аналогичная ситуация в `UnitDetail.tsx`.

## Что сделать

Переместить кнопку «Удалить» вниз страницы — в конец вкладки «Информация», после кнопки «Сохранить». Сделать её менее агрессивной визуально: `variant="outline"` с красным текстом вместо полностью красного фона.

### Файлы

| Файл | Изменение |
|------|-----------|
| `src/pages/PartnerDetail.tsx` | Убрать кнопку из заголовка (строки 144-148). Добавить её после кнопки «Сохранить» (строка ~289) — `variant="outline"`, `className="text-destructive"`, только для `isAdmin` |
| `src/pages/UnitDetail.tsx` | Аналогично: убрать из заголовка (строка 168), перенести вниз формы рядом с «Сохранить» |

### Результат

- Заголовок чистый: стрелка назад + название
- Кнопка удаления внизу, неброская, доступна только админу
- Кнопка сохранения и удаления рядом, но удаление визуально вторична
