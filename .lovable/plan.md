

# Исправить кнопку удаления и добавить кнопку просмотра

## Проблема
1. Кнопка удаления (корзина) в карточке портфолио имеет синий фон — передаётся `variant="default"`, что даёт primary-цвет. Нужно `variant="ghost"`, чтобы кнопка была прозрачной как соседняя кнопка редактирования.
2. Нет кнопки просмотра — для быстрого просмотра деталей без входа в редактирование.

## Изменения

### 1. Кнопка удаления — `UnitContactDetail.tsx`, строка 639
Заменить `variant="default"` на `variant="ghost"` в `ConfirmDialog` для портфолио-элемента. Иконка останется красной через `triggerClassName`.

### 2. Кнопка просмотра — `UnitContactDetail.tsx`, строки 633–641
Добавить кнопку с иконкой `Eye` перед кнопкой редактирования. По клику открывать read-only диалог с полной информацией:
- Все поля карточки (название, тип, год, источник, DOI, OA-статус, PDF, arXiv, том/номер/страницы, аннотация, тема, ключевые слова, издательство, ISBN, конференция, книга)
- Диалог без возможности редактирования — просто отображение данных

### 3. Новый диалог просмотра
Добавить `Dialog` для просмотра с read-only полями. Использовать простую структуру: заголовок + список пар «метка: значение». Показывать только заполненные поля.

### Затронутые файлы
- `src/pages/UnitContactDetail.tsx` — fix variant, добавить кнопку Eye и диалог просмотра
