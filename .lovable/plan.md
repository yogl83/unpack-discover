Единственное нереализованное требование — **визуальная матрица покрытия** в UI синка, показывающая режим каждой таблицы.

### Изменения

- В секции "Таблицы по умолчанию" и "Ручная синхронизация" рядом с каждой таблицей показать бейдж режима:
  - `ext` → `create + update` (зелёный) — для `partners`, `miem_units`
  - остальные → `update only` (серый)
- Заменить текущий бейдж `ext` на более информативный

- Добавить аналогичные бейджи режима рядом с чекбоксами таблиц
