

# Реорганизация меню по смысловым блокам

## Структура

```text
──────────────────────
⊞ Дашборд
──────────────────────
  ПАРТНЁРСТВА
──────────────────────
🏢 Партнеры
📋 Задачи партнеров
📇 Внешние контакты
──────────────────────
  МИЭМ
──────────────────────
👥 Коллективы МИЭМ
🧠 Компетенции
📇 Внутренние контакты
──────────────────────
  РАБОТА
──────────────────────
💡 Гипотезы
→  Следующие шаги
──────────────────────
⚙ Администрирование    (admin)
```

## Что меняется

| Аспект | Было | Станет |
|--------|------|--------|
| Структура | 1 группа, плоский список | 4 группы с `SidebarGroupLabel` |
| Контакты | Вложенный collapsible | Разделены: внешние → «Партнёрства», внутренние → «МИЭМ» |
| Дашборд | В общем списке | Отдельно сверху, без заголовка группы |
| Админка | В общем списке | Отдельная группа внизу |
| Визуальное разделение | Нет | Заголовки групп мелким шрифтом |

## Файл

| Файл | Действие |
|------|----------|
| `src/components/AppSidebar.tsx` | Убрать Collapsible контактов. Разбить на 4 `SidebarGroup`: Дашборд (без label), «Партнёрства» (Партнеры, Задачи, Внешние контакты), «МИЭМ» (Коллективы, Компетенции, Внутренние контакты), «Работа» (Гипотезы, Шаги). Админ — отдельная группа внизу. Плоские пункты, без вложенности. |

Маршруты, роли и бэкенд не затрагиваются.
