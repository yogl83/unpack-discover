

# Исправление верстки PDF-экспорта профайла

## Проблемы и решения

### Критичные

**1. Шапка пересекает заголовок (стр. 1)**
Сейчас `bodyTop = MARGIN + HEADER_H = 15 + 12 = 27mm`, а заголовок рисуется в `y = bodyTop = 27mm`. Колонтитул рисуется на `MARGIN + 4 = 19mm`, линия на `MARGIN + HEADER_H - 2 = 25mm`. Заголовок 18pt начинается на 27mm — слишком близко к линии 25mm. Решение: увеличить `HEADER_H` с 12 до 16mm, чтобы между линией шапки и контентом был запас.

**2. Таблица наезжает на шапку (стр. 2)**
`autoTable` использует `startY: y`, но не учитывает верхний колонтитул при переносе на новую страницу. Решение: передать `margin.top = MARGIN + HEADER_H` в autoTable, чтобы при переносе таблица начиналась ниже шапки.

**3. Разрыв таблицы — одна строка на новой странице**
autoTable сам разбивает таблицы. Добавить `showHead: 'everyPage'` для повтора заголовков и `pageBreak: 'avoid'` если таблица маленькая (≤6 строк), иначе `pageBreak: 'auto'`.

**4. Продолжение раздела без заголовка на новой странице**
При вызове `ensureSpace` внутри `renderTextBlock`, если происходит перенос на новую страницу — нужно отслеживать текущую секцию и рисовать «{название} (продолжение)» вверху новой страницы. Реализовать через колбэк `onPageBreak` который запоминает текущую секцию.

### Средней важности

**5. Escaped скобки `\[N\]`**
Текущий regex `\\?\[(\d+)\]\\?` не ловит все паттерны. Усилить: добавить дополнительный `.replace(/\\\[/g, "[").replace(/\\\]/g, "]")` как fallback после основного regex.

**6. Пропущен пробел после двоеточия**
Добавить в `preprocessText`: `.replace(/:([^\s])/g, ": $1")` — вставляет пробел после двоеточия, если его нет.

**7. Плотность текста**
Увеличить `LINE_H` с 4.8 до 5.2mm, `PARA_GAP` с 3 до 4mm. Добавить дополнительный gap между абзацами (двойной перенос строки → больший отступ).

**8. Ребаланс колонок таблицы**
В autoTable добавить `columnStyles: { 0: { cellWidth: 'auto', minCellWidth: 40 } }` и `tableWidth: 'auto'` с `cellPadding: 3` вместо 2.

**9. Блок «Источники» — визуальная иерархия**
- Увеличить шрифт номера+названия до 8.5pt (сейчас 8)
- URL — 7.5pt вместо 7
- Цитаты — 7pt, но цвет `rgb(80,80,80)` вместо `rgb(100,100,100)` для контрастности
- Добавить тонкую линию-разделитель между источниками
- Увеличить вертикальный зазор между источниками с 2mm до 4mm

**10. Неравномерные отступы**
Унифицировать: после секции — 8mm (сейчас 6), перед списком — 2mm gap, между bullet-ами — 1mm.

### Желательные

**11. Усилить титульную зону**
На первой странице: заголовок 20pt, подзаголовок «Профайл партнёра» 11pt серым, версия+дата. Затем если есть `summary_short` — вывести его как вводный абзац курсивом перед разделительной линией и основными секциями.

**12. Унификация списков**
Bullet-ы всегда `•`, нумерованные `1.`, `2.` — с одинаковым отступом `BULLET_INDENT = 7mm`. Для вложенных (если есть) — двойной отступ.

**13. Футер контрастнее**
Цвет `rgb(120,120,120)` вместо `rgb(150,150,150)`. Добавить дату генерации в левый угол футера.
